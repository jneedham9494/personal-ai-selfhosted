# Current System State

> Overall system specification for Personal AI Assistant

## System Overview

The Personal AI Assistant is a knowledge management system that connects Claude AI with Telegram for chat and Obsidian for storage. It provides:

- **Conversational AI** - Chat with Claude via Telegram
- **Knowledge Persistence** - Auto-save conversations to Obsidian
- **Project Tracking** - Manage projects and goals with progress
- **Proactive Reminders** - Smart nudging based on patterns
- **Secure Credentials** - 1Password integration

## Data Flow

### Conversation Flow
1. User sends message → Telegram
2. TelegramBot receives → adds to context
3. Context sent → ClaudeService
4. Response generated → sent to user
5. Conversation saved → ObsidianService
6. File created → Vault folder

### Nudging Flow
1. Cron job triggers
2. Recent conversations fetched
3. Patterns analyzed
4. Nudges generated
5. Rate limit checked
6. Notification sent → Telegram

## Known Limitations

### Security
- Memory-based user authorization
- Shell injection risk in 1Password service
- No rate limiting per user

### Scalability
- In-memory data structures
- File-based storage only
- Single process

### Reliability
- No retry logic
- No persistent state
- No health checks

### Observability
- Console logging only
- No metrics
- No alerting
