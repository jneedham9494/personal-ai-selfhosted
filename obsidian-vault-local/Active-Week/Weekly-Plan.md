## 🚨 Emergency Protocols

**If you want to quit a goal:**
1. STOP - Don't decide now
2. Check: How many hours invested? ___/100
3. If <100 hours: Cannot quit (Pre-Commitment Contract)
4. Read: [[Reading/Combating-Competitive-Quitting-Pattern#72-Hour-Rule]]
5. Wait 72 hours before reconsidering
6. Ask: Is this bottom-half anxiety or legitimate issue?

**If you feel behind:**
1. Behind WHOM? (If answer = others, this is Type 3 pattern)
2. Compare: Am I further than Monday Me? (If yes = progress)
3. Read: [[Reading/Enneagram-Report#Type-3-Comparison-Trap]]
4. Remember: Week 1 = showing up, not performing

**If overwhelmed:**
1. Pick ONE goal for today only
2. Do smallest possible version of task
3. Log the time
4. Tomorrow is a fresh start
5. Read: [[Reading/Q4-Success-Framework#Overwhelm-Protocol]]
