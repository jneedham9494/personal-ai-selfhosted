## The Path to Stage

### Phase 1: Preparation (Weeks 1-2)
**Consume Comedy:**
- [ ] Watch 5+ hours of standup specials
- [ ] Study favorite comedians' structure
- [ ] Listen to comedy podcasts (WTF, The Honeydew)
- [ ] Read "The Comedy Bible" or similar
- [ ] Understand setup → punchline → tag structure

**Find Local Scene:**
- [ ] Research local open mics (comedy clubs, bars)
- [ ] Find comedy Facebook groups
- [ ] Locate comedy classes (if interested)
- [ ] Identify regular show nights
- [ ] Note sign-up procedures

**Start Writing:**
- [ ] Buy dedicated comedy notebook
- [ ] Write 5 joke premises per day
- [ ] Mine personal experiences for material
- [ ] Practice "funny filter" in daily life
- [ ] Don't judge, just write

### Phase 2: Writing & Refinement (Weeks 3-4)
**Develop Material:**
- [ ] Write about what you know (authentic = funny)
- [ ] Find your comedic perspective/voice
- [ ] Turn observations into jokes
- [ ] Write setup and punchline for 20+ jokes
- [ ] Add tags (extra laughs after punchline)

**Attend Open Mics:**
- [ ] Go as audience 2-3 times first
- [ ] Study room dynamics
- [ ] Notice what works/doesn't
- [ ] Talk to other comedians
- [ ] Learn the vibe and format

**Prepare First Set:**
- [ ] Select best 8-10 jokes
- [ ] Arrange in logical order
- [ ] Write out transitions
- [ ] Time yourself (aim for 3-5 minutes)
- [ ] Practice in front of mirror
- [ ] Record yourself on phone

### Phase 3: First Performances (Weeks 5-8)
**First Open Mic:**
- [ ] Sign up for spot (arrive early!)
- [ ] Bring supporters (2-3 friends)
- [ ] Expect to be nervous (everyone is)
- [ ] Perform your set
- [ ] Record audio (if allowed)
- [ ] Celebrate - YOU DID IT!

**After First Performance:**
- [ ] Review recording
- [ ] Note what worked
- [ ] Identify what to cut
- [ ] Don't be too hard on yourself
- [ ] Write new material immediately

**Repeat Performances:**
- [ ] Perform at least 1x per week
- [ ] Try different venues
- [ ] Adjust material based on response
- [ ] Build stage confidence
- [ ] Track which jokes land

### Phase 4: Development (Months 3-6)
**Build Your Set:**
- [ ] Develop signature bits
- [ ] Create callback opportunities
- [ ] Work on crowd work skills
- [ ] Handle silence/bombing gracefully
- [ ] Develop stage presence

**Continuous Improvement:**
- [ ] Record every performance
- [ ] Take notes immediately after
- [ ] Kill jokes that don't work (murder your darlings)
- [ ] Polish jokes that get laughs
- [ ] Add new material constantly

**Get Feedback:**
- [ ] Ask experienced comics for advice
- [ ] Share material with comedy friends
- [ ] Notice audience reactions
- [ ] Adjust based on data
- [ ] Trust the process

## Writing System

### Daily Writing Practice (15-30 min)
**Morning Pages:**
- [ ] Write 3 pages stream of consciousness
- [ ] Mine for funny observations
- [ ] No filter, no judgment

**Joke Writing:**
- [ ] Target: 5 premises per day
- [ ] Use premise → setup → punchline format
- [ ] Don't need to be "done" - just written

**Sources for Material:**
- Personal embarrassments (gold!)
- Weird observations
- Relationships and dating
- Family dynamics
- Work/career frustrations
- Current events (be careful)
- Your specific perspective/identity

### Joke Structure Templates

**Setup → Punchline:**
```
"I [observation/situation]...
[unexpected twist/punchline]"
```

**Rule of Three:**
```
"I tried [A], then [B], then [C - absurd]"
```

**Misdirection:**
```
"I thought [expected thing]...
turns out [unexpected reality]"
```

**Callback:**
```
Reference earlier joke later in set
Shows structure, big laughs
```

## Performance Skills

### Stage Presence Checklist
- [ ] Make eye contact
- [ ] Use microphone properly (3-4 inches)
- [ ] Don't pace (plant feet)
- [ ] Speak clearly and project
- [ ] Pause for laughs (don't talk over them)
- [ ] Stay committed to material
- [ ] Own mistakes/bombing

### Dealing with Bombs
- [ ] Everyone bombs - it's normal
- [ ] Learn from it, don't dwell
- [ ] Material might need work
- [ ] Wrong room/audience
- [ ] Timing was off
- [ ] Keep going to next joke

### Handling Hecklers
- [ ] Don't engage at open mics (time is limited)
- [ ] Have 2-3 comeback lines ready
- [ ] Stay in control
- [ ] Most audiences are on your side
- [ ] Kill them with kindness or wit

## Mindset & Motivation

### Remember:
- **Everyone starts bad** - Your heroes bombed hundreds of times
- **Stage time > everything** - Writing is important, but stage time is how you learn
- **Bombing is data** - Not personal failure, just information
- **Be authentic** - Your unique perspective is your strength
- **Comedy is a muscle** - Gets stronger with practice
- **Have fun** - If you're not enjoying it, audience won't either
- **Be patient** - It takes 5-7 years to "find your voice"

- "I deserve to be on stage"

- "My perspective is valuable"
